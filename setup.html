Настройка Firebase
Одноразовый запуск • Заполнение базы
Перейти в админку →
1
Firebase Console

Создай Firestore (если ещё нет)

Build → Firestore Database → Create database → Start in test mode → eur3 → Enable. Если Firestore уже создан — пропусти.

2
Firebase Console → Firestore → Rules

Правила безопасности

Вставь эти правила и нажми Publish:

rules_version = '2'; service cloud.firestore { match /databases/{database}/documents { match /{document=**} { allow read, write: if true; } } }
3
Автоматически

Заполнить каталог предметов и шаблоны заданий

Создаст 4 предмета (subject_catalog) и шаблоны заданий (task_templates). Если данные уже есть — не перезапишет.

Нажми «Проверить состояние» чтобы начать…
4
Необязательно

Заполни шаблоны заданий

Здесь можно вписать описание, конспект, домашку и подсказки для каждого шаблона. Данные подтянутся ко всем ученикам, которым будет добавлен этот предмет в дальнейшем.

5
Готово

Перейди в админку и создай первого ученика

После заполнения базы открой админку, создай ученика, выбери предмет — задания подтянутся автоматически.
